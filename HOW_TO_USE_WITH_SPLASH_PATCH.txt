HOW TO USE

1. Download all parts.
2. Extract all zips into the same parent folder.
3. When asked to merge folders, choose merge/replace.
4. The merged output should contain one Calamity_War folder.
5. Copy/merge that Calamity_War folder into the real playable GitHub repo.

SPLASH SCREEN PATCH

The Emma Woods Studio splash files are included in PART 00:

To make the splash actually run, the real playable index.html needs these two lines:

<link rel="stylesheet" href="patches/splash_screen/emma_woods_splash.css">
<script src="patches/splash_screen/emma_woods_splash.js"></script>

If the playable game files are in a subfolder, adjust those paths to match the final folder structure.
